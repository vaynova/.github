# Vaynova

Vaynova is a deep-tech company building industrial edge-AI semiconductors. Our mission is to enable sub ‑watt inference silicon for domain-specific condition monitoring.

## Mission
We create ultra-low-power hardware for real-time anomaly detection on vibration and other 1 ‑D sensor signals in rotating machinery and industrial assets. Our first product, EdgeSense‑1W, combines a DS‑Conv1D compute core, SRAM-first memory hierarchy, and minimal host interface to deliver sub 1 W inference for predictive maintenance.

## Repositories
- **edgesense ‑1w ‑architecture** – public architectural narrative and interface specification for the EdgeSense ‑1W inference co ‑processor.
- **edgesense ‑reference ‑tooling** – reference tools such as register‑map generators and synthetic vibration simulators.
- **ds ‑conv1d ‑benchmarks** – comparisons of DS‑Conv1D with standard Conv1D and other models on vibration condition‑monitoring datasets.
- **condition‑monitoring‑edge‑survey** – a literature survey on edge AI for industrial condition monitoring (coming soon).
- **.github** – this repository holds our organisation profile and community files.
